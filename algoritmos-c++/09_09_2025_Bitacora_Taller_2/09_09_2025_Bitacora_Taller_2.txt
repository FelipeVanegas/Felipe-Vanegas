- pwd: La terminal muestra la ubicación.
- ls: Muestra los elementos en la ubicción actual.
- mkdir: Crear una nueva carpeta.
- cd: Ingresar a un archivo o carpeta.
- ls-1: Muestra los archivos en un formato detallado.
- touch: Crea un archivo dentro del directorio (archivo de texto).
- cp: Copiar un archivo a otro.
- mv: Cambiarle el nombre a un archivo.
- rm: Comando para eliminar.
- cd: Regresar dentro de la tertminal.
- rm -r: Eliminar todo el contenido.
- find: Buscar archivo.
- clear: Limpiar el terminal.

Ventajas de usar la terminal:

La terminal tiene ofrece ventajas como rapidez, control total sobre el sistema y menor consumo de recursos en comparación con la interfaz gráfica. Permite automatixar tareas mediante scripts, acceder de forma remota a servidores y aprender mas sobre el sistema operativo. Además, ofrece frexibilidad y personalización para tareas avanzadas.
